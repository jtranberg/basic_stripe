There is two folders. 
Client is the front end.
Server is your backend (engine).

add .env file to root of client and root of server

do not change key names as they match the internal code
add from stripe 
STRIPE_PUBLISHABLE_KEY=pk_test_...your keys
STRIPE_SECRET_KEY=sk_test_your key

npm install
get dependancies

cd server http-server
cd client npm run start
